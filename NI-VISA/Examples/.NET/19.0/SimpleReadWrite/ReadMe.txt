Example Title:           Simple Read/Write

Category:                NI-VISA

Description:             This example demonstrates how to use the NI-VISA .NET Library to
                         perform simple synchronous I/O operations.  You can use this example
                         to send commands, such as "*IDN?\n", to your instrument.

Language:                Visual C#

Required Software:       NI-VISA

Required Hardware:       Any message-based device
