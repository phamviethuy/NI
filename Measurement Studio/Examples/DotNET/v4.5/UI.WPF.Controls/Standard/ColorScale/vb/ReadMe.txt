Example Title:		ColorScale
                         
Example Filename:	ColorScale.sln
                         
Category:			User Interface - Intensity Graph
                         
Description:	This example demonstrates how to customize the Color Scale and Color Markers. You can click on any of the radio buttons to change the color marker settings. In this example, the data plotted remains the same, but the color scale properties (ScaleKind, LowColor, HighColor, Markers, IsInterpolated) change. These properties can be chaged under 'Custom Color Scale Properties'. Use 'Add Color Marker' to customize the provided color markers. 
                                       
Software Group:		Measurement Studio

Required Software:	.NET Framework 4.0 and Microsoft Visual Studio 2010.                     
                         
Language:			Visual Basic .NET, Visual C#

Language Version:	.NET 4.0
