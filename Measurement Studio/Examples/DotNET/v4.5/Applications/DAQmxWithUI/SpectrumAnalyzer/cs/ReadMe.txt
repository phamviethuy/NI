Example Title:           Spectrum Analyzer
                         
Example Filename:        Spectrum Analyzer.sln
                         
Category:                Application - DAQ, Analysis
                         
Description:             This example acquires a waveform from a Data Acquistion device (using NI-DAQ driver), and computes the averaged, windowed power spectrum. It then graphs the power spectrum in the desired units and it allows the user to specify the window and scale.

Required Software:

Language Version:        7.0
                         
Hardware Group:          Data Acquisition
                         
Driver Name:             NI-DAQmx 
                         
Driver Version:          8.1 or later
                         
Required Hardware:       Analog Input DAQ Device
