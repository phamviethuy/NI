Example Title:           Error Data Modes
                         
Example Filename:        ErrorDataModes.sln
                         
Category:                User Interface - 2D Graph - Error Bands
                         
Description:             This example demonstrates basic use of error data modes in the 2D Graph.

                         The radio buttons select different error data modes for the plot.  None 
                         does not generate any error data; Constant generates constant offsets 
                         for the plot data; Percent generates percent offsets for the plot data.

                         The drop-down box selects different display modes for the error data.
                         None hides the error data; High shows error data in the direction of 
                         increasing axis values; Low shows error data in the direction of 
                         decreasing axis values; HighAndLow shows all error data.
