Example Title:           Charting
                         
Example Filename:        Charting.sln
                         
Category:                User Interface - 2D Graph
                         
Description:             This example demonstrates charting in the 2D Graph. The radio buttons in 				 
			 the Charting Settings group box control the chart style of the 			 
			 axis. Set the radio button to Scope Chart to scroll the axis like a scope.				 
			 Set the radio button to Strip Chart to scroll the axis to the most recent 				 
			 data value. The Vertical checkbox charts the data vertically using the 				 
			 chart style specified by the radio buttons.
