Example Title:              ColorScale
                         
Example Filename:     ColorScale.sln
                         
Category:                      User Interface - Intensity Graph
                         
Description:                 This example demonstrates how to customize the Color Scale and Color Map. You can click on any of the radio buttons to change the color map settings. In this example, the data plotted remains the same, but the color scale properties (Range, LowColor, HighColor, ColorMap) change. Click  the "Edit Color Map" button to customize the selected color map. 
                                       
Software Group:          Measurement Studio                          
                         
Language:                     Visual C# .NET
