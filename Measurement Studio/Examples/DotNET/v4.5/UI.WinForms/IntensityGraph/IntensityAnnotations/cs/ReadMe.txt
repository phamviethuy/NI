Example Title:              IntensityAnnotations
                         
Example Filename:     IntensityAnnotations.sln
                         
Category:                     User Interface - Intensity Graph
                         
Description:                  This example demonstrates how to use annotations to highlight the minimum and maximum values of the plotted intensity data. Click the "Generate Data" button to plot new data. You can use the radio buttons to hide and show different parts of the annotation.
                                       
Software Group:          Measurement Studio
                         
Language:                     Visual C# .NET
