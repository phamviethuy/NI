Example Title:		Simple Numeric Edit
                         
Example Filename:	SimpleNumericEdit.sln
                         
Category:		UI
                         
Description:		This example demonstrates some of the features of the Numeric
                        Edit control.  You can change the interaction modes, the range,
                        and the coercion properties of the Numeric Edit by adjusting
                        the values on the dialog.  Check the "Connect to Knob" check box
                        to connect the Numeric Edit to the Knob.
